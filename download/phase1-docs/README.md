# InvestPro Platform

A managed investment plan platform that enables users worldwide to deposit funds into tiered investment plans with fixed return schedules. The platform supports both cryptocurrency and gift card deposits, operates in dual demo and live modes simultaneously, and provides a referral-driven growth model with binary bonus structures. Built for global deployment with multi-language support and a premium dark-theme user experience.

## Tech Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Backend:** Node.js, REST API
- **Database:** PostgreSQL with Prisma ORM
- **Caching:** Redis
- **Media:** Cloudinary (image storage and transformation)
- **Email:** Resend + React Email
- **Deployment:** Docker, Docker Compose, Coolify
- **Language:** TypeScript (full-stack)
